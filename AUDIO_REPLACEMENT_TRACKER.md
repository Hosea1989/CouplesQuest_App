## Replacement Workflow

1. Find or create a sound
2. Export/convert to **WAV, 44100 Hz, 16-bit, mono** (use Audacity if needed)
3. Rename to the **exact filename** from the table above
4. Drop it into `CouplesQuest/Resources/Sounds/` replacing the old file
5. Build in Xcode -- it picks up automatically
6. Check the box in this file!

**Tip**: In the app, go to **Settings > Sound Effects** to hear your sounds as you test them.
